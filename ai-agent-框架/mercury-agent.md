## 项目简介
灵魂驱动的AI Agent，具有权限硬化工具、token预算和多渠道访问，可24/7从CLI或Telegram运行。

## 使用场景
- 构建7x24小时运行的AI助手
- 多渠道（CLI/Telegram）AI Agent
- 带权限控制的自主Agent

## 方法
- 权限硬化工具系统，确保Agent操作安全可控
- Token预算管理，防止资源过度消耗
- 多渠道访问支持（CLI、Telegram），统一交互体验
- 灵魂驱动的人格架构，赋予Agent一致的个性表达
- 24/7守护进程模式，持续运行不中断

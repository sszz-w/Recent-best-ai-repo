## 项目简介
Claude Code 的开源替代方案，可在任何环境运行并支持任意 LLM 提供商。模型无关的 AI 编码智能体。

## 使用场景
- 使用任意 LLM（OpenAI、DeepSeek、本地模型）获得类似 Claude Code 的体验
- 自托管 AI 编码智能体
- 构建自定义 AI 开发工作流

## 方法
- 提供商无关的 LLM 抽象层
- 基于 CLI 的智能体循环
- 兼容多种模型的工具调用框架
- TypeScript 运行时

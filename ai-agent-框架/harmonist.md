# Harmonist

## 基本信息
- **GitHub**: https://github.com/GammaLabTechnologies/harmonist
- **Stars**: 1,668
- **语言**: Python
- **创建时间**: 2026-04-23

## 项目简介
便携式AI Agent编排系统，机械协议强制执行，186个Agent，零运行时依赖。

## 使用场景
- 大规模多Agent编排
- 零依赖的Agent框架部署
- 协议驱动的Agent协作

## 方法
- 机械协议强制执行，确保Agent行为可预测
- 内置186个预构建Agent，开箱即用
- 零运行时依赖，便携部署无负担
- 基于提示工程的编排机制，灵活可扩展

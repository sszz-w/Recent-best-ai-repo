## 项目简介
Google官方CLI工具和技能集，将任何编码助手变为创建、评估和部署Google Cloud AI Agent的专家。

## 使用场景
- 在Google Cloud上部署AI Agent
- 使用Gemini构建企业级Agent
- AI Agent的评估和测试

## 方法
- Google ADK (Agent Development Kit) integration
- Gemini model support
- Google Cloud deployment
- Agent evaluation framework

## 项目简介
Claude + Obsidian知识伴侣。基于Karpathy的LLM Wiki模式，构建持续积累的wiki知识库。

## 使用场景
- 用AI自动构建和维护Obsidian知识库
- 实现持续积累的第二大脑
- AI辅助的知识研究和整理

## 方法
- 实现Karpathy LLM Wiki模式，知识持续积累而非一次性对话
- 提供/wiki、/save、/autoresearch等命令，简化知识管理操作
- 持久化复合知识库，知识随时间不断丰富和关联
- Claude Code skill集成，可在开发环境中直接使用

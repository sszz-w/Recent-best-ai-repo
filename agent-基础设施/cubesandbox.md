# CubeSandbox

## 基本信息
- **GitHub**: https://github.com/TencentCloud/CubeSandbox
- **Stars**: 5,620
- **语言**: Rust
- **创建时间**: 2026-04-10

## 项目简介
腾讯云开源的即时、并发、安全、轻量级AI Agent沙箱环境。为AI Agent提供安全隔离的代码执行能力。

## 使用场景
- 为AI Agent提供安全的代码执行环境
- 隔离运行不可信的Agent生成代码
- 大规模并发Agent任务执行

## 方法
- 基于Rust实现的轻量级容器，保证高性能和内存安全
- 即时启动，实现低延迟的代码执行
- 并发沙箱管理，支持大规模Agent任务并行
- 安全隔离机制，防止不可信代码对宿主系统的影响

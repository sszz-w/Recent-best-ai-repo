# Hermes Web UI

- **Stars**: 4,853

## 项目简介
Hermes Agent的Web管理面板——多平台AI对话、会话管理、定时任务和使用量分析。

## 使用场景
- 多模型AI对话管理界面
- Agent会话管理和调度
- AI使用量分析和监控

## 方法
- 基于Vue3 + TypeScript构建的Web仪表盘
- 多模型支持，统一管理不同LLM的对话
- 会话管理系统，支持历史记录和上下文维护
- 定时任务执行，支持Agent自动化调度
- 使用量分析追踪，监控API调用和资源消耗

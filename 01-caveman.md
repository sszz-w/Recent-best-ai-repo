## 项目简介
Claude Code 技能插件，通过模仿"穴居人"式简洁语言风格，将 token 消耗降低 65%。这是一种提示工程技术，通过语言压缩来减少 token 使用量。

## 使用场景
- 在使用 Claude Code 时降低 API 调用成本
- 在长时间编码会话中优化 token 消耗
- 节省 LLM API 调用费用

## 方法
- 自定义系统提示词，强制生成简洁的"穴居人"风格回复
- 基于 Skill 机制与 Claude Code 集成
- 通过语言压缩实现 token 优化

## 项目简介
基于 Claude Code 构建的 AI 驱动求职系统。包含 14 种技能模式、Go 语言仪表盘、PDF 生成和批处理功能。

## 使用场景
- 自动化求职申请和简历定制
- 使用 AI 教练进行面试准备
- 通过仪表盘追踪求职进度

## 方法
- 基于 Claude Code Skill 架构，包含 14 个专业模式
- Go 语言实现的 Web 仪表盘
- PDF 生成管线
- 支持多份申请的批处理机制

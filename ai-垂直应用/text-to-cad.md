# Text-to-CAD

## 基本信息

- **Stars**: 2,797

## 项目简介
用自然语言生成CAD模型的Agent技能集合，支持机器人设计和硬件设计。

## 使用场景
- 用自然语言描述生成3D CAD模型
- AI辅助的机器人和硬件设计
- 快速原型设计

## 方法
- Agent skills for CAD generation
- Supports STEP, STL, URDF, SDF formats
- Build123d + OpenCascade backend
- WASM-based browser rendering
